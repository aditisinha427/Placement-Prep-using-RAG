# 🚀 Placement Prep Copilot (RAG-based AI Assistant)

## 📌 Overview
An AI-powered interview assistant that answers questions from Operating Systems, DBMS, and Computer Networks using **Retrieval-Augmented Generation (RAG)**.

---

## ⚙️ Tech Stack
- Python  
- FastAPI  
- Streamlit  
- FAISS (Vector Database)  
- Gemini API  
- LangChain  

---

## 🧠 Features
- 🔍 Semantic search over structured notes  
- 🤖 Context-aware AI responses  
- ⚡ REST API using FastAPI  
- 💻 Interactive UI using Streamlit
